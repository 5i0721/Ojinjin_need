# 禁禁你的 | 群聊禁言拓展插件（Ojinjin_need）

> 提供通过指定指令禁言的群聊管理插件，纯中文指令，可扩展可配置。

### 📝 最新更新：v2.0.2

- 添加解禁功能

- 如有问题可添加QQ群讨论：729827554

## ✨ 功能概览

| 指令     | 格式                | 说明                                           |
| -------- | ------------------- | ---------------------------------------------- |
| **禁言** | `禁言 @用户 [时长]` | 对指定用户执行禁言，不填时长默认60秒           |
| **鞭尸** | `鞭尸 @用户 [次数]` | 对指定用户反复执行 禁言30天→解禁 循环，默认5次 |
| **解禁** | `解禁 @用户`        | 对指定用户执行解除禁言                         |

## 📦 安装

将插件文件夹 `Ojinjin_need` 放入 MaiBot 的 `plugins/` 目录下，重启即可自动加载。

首次启动后会在插件目录下自动生成 `config.toml` 配置文件。

## 🎮 使用方式

### 禁言指令

```
禁言 @用户 10        → 禁言10秒
禁言 @用户 10m       → 禁言10分钟
禁言 @用户 1h        → 禁言1小时
禁言 @用户 1d        → 禁言1天
禁言 @用户           → 禁言60秒（默认）
```

**时长单位说明：**

| 单位         | 含义 | 示例                 |
| ------------ | ---- | -------------------- |
| 纯数字 / `s` | 秒   | `60` 或 `60s` = 60秒 |
| `m`          | 分钟 | `10m` = 10分钟       |
| `h`          | 小时 | `2h` = 2小时         |
| `d`          | 天   | `1d` = 1天           |

### 鞭尸指令

```
鞭尸 @用户           → 反复禁言/解禁 5次（默认），最终禁言30天
鞭尸 @用户 20        → 反复禁言/解禁 20次，最终禁言30天
```

**鞭尸执行流程：**

```
禁言30天 → 等待 → 解禁 → 等待 → 禁言30天 → 等待 → 解禁 → ...（循环N次）→ 最终禁言30天
```

> ⚠️ 鞭尸会产生大量API调用，间隔太短可能被平台风控，请合理设置间隔时间。

### 解禁指令

```
解禁 @用户           → 解除禁言
```

### 指令格式容错

插件对格式非常宽容，以下写法都能正确识别：

```
禁言 @用户 10
禁言@用户 10
禁言  @用户  10m
禁言@用户10
鞭尸 @用户
鞭尸@用户 20
鞭尸  @用户  20
解禁
@用户
```

## ⚙️ 配置文件说明

首次启动后自动生成 `config.toml`，以下是完整配置项解析：

### 插件基本配置 `[plugin]`

| 配置项           | 类型   | 默认值    | 说明                       |
| ---------------- | ------ | --------- | -------------------------- |
| `enabled`        | bool   | `true`    | 是否启用插件               |
| `config_version` | string | `"2.0.1"` | 配置文件版本，请勿手动修改 |

### 群聊过滤 `[group_filter]`

控制哪些群可以使用本插件的指令。

| 配置项       | 类型   | 默认值        | 说明                 |
| ------------ | ------ | ------------- | -------------------- |
| `mode`       | string | `"whitelist"` | 过滤模式，见下方说明 |
| `group_list` | list   | `[]`          | 群号列表             |

**过滤模式说明：**

| 模式                | 列表为空时    | 列表有值时                  |
| ------------------- | ------------- | --------------------------- |
| `whitelist`（默认） | ✅ 所有群可用 | ✅ 只有列表中的群可用       |
| `blacklist`         | ❌ 所有群禁用 | ❌ 列表中的群禁用，其余可用 |

**配置示例：**

```toml
[group_filter]
# 只允许这两个群使用
mode = "whitelist"
group_list = ["123456789", "987654321"]
```

```toml
[group_filter]
# 禁止这个群使用，其余群都可以
mode = "blacklist"
group_list = ["666666666"]
```

```toml
[group_filter]
# 所有群都可以用（白名单 + 空列表）
mode = "whitelist"
group_list = []
```

### 用户过滤 `[user_filter]`

控制哪些用户可以使用指令。**未通过过滤的用户触发指令时不会在聊天框显示任何内容**，仅在后台日志中记录。

| 配置项      | 类型   | 默认值        | 说明                 |
| ----------- | ------ | ------------- | -------------------- |
| `mode`      | string | `"blacklist"` | 过滤模式，见下方说明 |
| `user_list` | list   | `[]`          | 用户QQ号列表         |

**过滤模式说明：**

| 模式                | 列表为空时    | 列表有值时                  |
| ------------------- | ------------- | --------------------------- |
| `blacklist`（默认） | ✅ 所有人可用 | ❌ 列表中的人禁用，其余可用 |
| `whitelist`         | ❌ 所有人禁用 | ✅ 只有列表中的人可用       |

**配置示例：**

```toml
[user_filter]
# 只允许管理员使用
mode = "whitelist"
user_list = ["123456789", "987654321"]
```

```toml
[user_filter]
# 所有人都能用（黑名单 + 空列表，这是默认配置）
mode = "blacklist"
user_list = []
```

```toml
[user_filter]
# 禁止某个用户使用
mode = "blacklist"
user_list = ["666666666"]
```

### 禁言功能配置 `[mute]`

| 配置项             | 类型 | 默认值    | 说明                               |
| ------------------ | ---- | --------- | ---------------------------------- |
| `default_duration` | int  | `60`      | 不指定时长时的默认禁言秒数         |
| `min_duration`     | int  | `1`       | 允许的最短禁言时长（秒）           |
| `max_duration`     | int  | `2592000` | 允许的最长禁言时长（秒），默认30天 |

### 鞭尸功能配置 `[whip]`

| 配置项             | 类型  | 默认值 | 说明                              |
| ------------------ | ----- | ------ | --------------------------------- |
| `default_count`    | int   | `5`    | 不指定次数时的默认鞭尸次数        |
| `max_count`        | int   | `100`  | 允许的最大鞭尸次数                |
| `interval_seconds` | float | `3.0`  | 每次禁言/解禁操作之间的间隔（秒） |

> ⚠️ `interval_seconds` 设置过小（如0.5秒以下）可能导致QQ平台风控，建议保持2-5秒。

## 📄 完整默认配置参考

```toml
[plugin]
enabled = true
config_version = "2.0.1"

[group_filter]
# whitelist=只有列表中的群可用（列表为空则所有群可用）
# blacklist=列表中的群禁用（列表为空则所有群禁用）
mode = "whitelist"
group_list = []

[user_filter]
# blacklist=列表中的用户禁用（列表为空则所有人可用）
# whitelist=只有列表中的用户可用（列表为空则所有人禁用）
mode = "blacklist"
user_list = []

[mute]
default_duration = 60
min_duration = 1
max_duration = 2592000

[whip]
default_count = 5
max_count = 100
interval_seconds = 3.0
```

## 🔧 前置要求

- MaiBot Core ≥ 0.7.0
- 机器人在目标群内拥有**管理员权限**（否则禁言命令会失败）

## ❓ 常见问题

**Q: 发了指令没有任何反应？**

- 检查机器人是否是群管理员
- 检查 `group_filter` 配置，当前群是否被允许
- 检查 `user_filter` 配置，你的QQ是否被允许（未通过过滤的用户不会收到任何回复）
- 查看后台日志，搜索 "禁言" 或 "鞭尸" 关键词

**Q: 鞭尸执行到一半失败了？**

- 可能是操作太快被平台风控，尝试增大 `interval_seconds`
- 检查机器人管理员权限是否被取消

**Q: 能禁言管理员或群主吗？**

- 不能，这是QQ平台的限制，机器人无法禁言同级或更高权限的用户
